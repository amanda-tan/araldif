Format


Choose dates from 1948-01-01 to 2010-12-31

Startdate: YYYY-MM-DD
Enddate: YYYY-MMM-DD
Stations (must be all caps):
ARYSS
HODJI
NARYN
UCHKU
TYUME
UTSKE

Example Calls


Get streamflow for Uchku gauge for January 01, 1995 to December 31, 2005 as a downloadable .csv file:
http://araldig.azurewebsites.net/api/gethydrograph?start=1995-01-01&end=2005-12-31&station=UCHKU

Get streamflow for Tyume gauge for November 15, 1965 to March 23, 1997 as a JSON serialized dump to screen (note that using Internet Explorer will bring this up in Visual Studio):
http://araldig.azurewebsites.net/api/gethydrograph?start=1965-11-15&end=1997-03-23&station=TYUME&jsondat=TRUE

To generate a plot, just add plot=TRUE to the end of the API
http://araldig.azurewebsites.net/api/gethydrograph?start=1995-01-01&end=2005-12-31&station=UCHKU&plot=TRUE
